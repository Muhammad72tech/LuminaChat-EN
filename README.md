# LuminaChat-EN (English version)

**Lightweight self-hosted chat, messenger, and chatroom platform built with PHP and SQLite.**

LuminaChat is a portable open-source messaging platform designed for private messaging, group chat, community communication, and local network collaboration. With zero installation and no database server requirements, it can be deployed within minutes on shared hosting, VPS servers, private infrastructure, or local networks.

Simply extract the project files and start chatting.

## Why LuminaChat?

Modern communication increasingly depends on centralized platforms and external services. LuminaChat provides a lightweight alternative that can be deployed and controlled by individuals, communities, schools, organizations, or groups of friends.

The goal is simple:

> Keep people connected through infrastructure they own and control.

Whether you need a private chatroom for friends, a self-hosted messenger for a community, a communication platform for a local network, or a resilient communication solution during service disruptions, LuminaChat is designed to remain simple, portable, and easy to deploy.

## Features

* Lightweight PHP and SQLite architecture
* Self-hosted messenger and chatroom
* No installation required
* No MySQL or external database server required
* Portable deployment from a ZIP archive
* Private messaging and group communication
* Community and local network friendly
* Minimal server requirements
* Open source and easy to customize
* Fast backup and migration

## Quick Start

1. Download the ZIP archive.
2. Extract it on your Linux server, VPS, web hosting account, or local network server.
3. Follow the instructions in `help.txt`.
4. Create your chatroom and invite your friends or community.
5. Remove `help.txt` after setup for additional security.

## Vision

LuminaChat is more than a simple chat application.

It is a lightweight communication platform built around the idea that people should be able to communicate without depending entirely on large centralized services. By combining portability, simplicity, and self-hosting, LuminaChat helps communities maintain communication using infrastructure they control.
